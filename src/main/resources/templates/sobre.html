Sobre o FeijoadaWeek

O principal objetivo deste site é, obviamente, que seja mais um motivo para comer feijoada.

O motivo secundário é usar para prática e experiências de programação.

Atualmente, este blog está feito em Java e mais explicações podem ser encontradas no repositório no Github:

https://github.com/ahrocha/feijoadaweek-site-spring

Aos poucos, além de conteúdos, o código e recursos deste site também vão evolur.

Espero que goste e se divirta com o conteúdo.
